# Politique de Confidentialité - IMMO_TN

**Dernière mise à jour: Novembre 2025**

## Introduction

IMMO_TN ("nous", "notre", "nos") exploite l'application mobile IMMO_TN. Cette page vous informe de nos politiques concernant la collecte, l'utilisation et la divulgation des données personnelles lorsque vous utilisez notre service.

## Informations que nous collectons

### Informations fournies par l'utilisateur
- **Compte utilisateur**: Nom, email, numéro de téléphone
- **Profil**: Photo de profil, biographie
- **Propriétés**: Informations sur les biens immobiliers que vous publiez
- **Messages**: Conversations avec d'autres utilisateurs

### Informations collectées automatiquement
- **Données de localisation**: Pour afficher les propriétés sur la carte
- **Données d'utilisation**: Pages visitées, fonctionnalités utilisées
- **Informations sur l'appareil**: Type d'appareil, système d'exploitation

## Comment nous utilisons vos informations

Nous utilisons les informations collectées pour:
- Fournir et maintenir notre service
- Améliorer l'expérience utilisateur
- Envoyer des notifications importantes
- Assurer la sécurité et prévenir la fraude
- Analyser l'utilisation de l'application

## Partage de vos informations

Nous ne vendons jamais vos informations personnelles. Vos données peuvent être partagées:
- **Avec d'autres utilisateurs**: Informations de profil et propriétés visibles publiquement
- **Avec nos fournisseurs de services**: Pour héberger et maintenir l'application
- **Pour des raisons légales**: Si requis par la loi

## Sécurité des données

Nous prenons la sécurité de vos données au sérieux:
- Chiffrement des mots de passe (bcrypt)
- Communication sécurisée (HTTPS)
- Authentification par token JWT
- Stockage sécurisé des données

## Vos droits

Vous avez le droit de:
- Accéder à vos données personnelles
- Corriger vos informations
- Supprimer votre compte
- Exporter vos données
- Retirer votre consentement

## Cookies et technologies similaires

Nous utilisons:
- **AsyncStorage**: Pour stocker les préférences localement
- **Tokens d'authentification**: Pour maintenir votre session

## Services tiers

Notre application utilise:
- **OpenAI**: Pour l'assistant IA (optionnel)
- **Expo**: Pour le développement mobile
- **Google Maps**: Pour les cartes interactives

## Modifications de cette politique

Nous pouvons mettre à jour cette politique périodiquement. Nous vous informerons de tout changement en publiant la nouvelle politique sur cette page.
